# HR Analytics Dashboard – Power BI

## Project Overview

This project presents an **HR Analytics dashboard** built in Power BI to analyze employee attrition, demographics, salary distribution, and workforce trends.

The dashboard helps identify:

* Attrition patterns across departments and job roles
* Salary distribution and its relationship with attrition
* Employee demographics such as age, gender, and education
* Workforce tenure and retention trends

## Key Metrics

* Total Employees: 1480
* Average Age: 37
* Average Salary: 6.5K
* Average Years at Company: 7
* Attrition Count: 238
* Attrition Rate: 16%

## Tools Used

* Power BI
* Data Modeling
* DAX Calculations
* Interactive Dashboard Design
